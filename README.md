# SPOTIFY-CLONE
A static Spotify web player clone built using HTML and CSS .

A responsive clone of the Spotify web player interface built using HTML and CSS. This project focuses on recreating the core layout and visual design of Spotify, including a fixed sidebar, scrollable main content area, dark-themed styling, and card-based sections like Recently Played and Trending. Built to strengthen frontend fundamentals and understand real-world UI structuring using modern CSS techniques.
